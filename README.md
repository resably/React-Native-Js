
# React Native JS with reduxtoolkit-tailwindcss

React native JS version for quick start.




## Includes

- Redux Toolkit
- React Navigation
- TailwindCSS
- React Native Vector Icons
- Expo
- Axios



  
## Project Structuring

```bash
├── App.js
├── app.json
├── source
│   ├── nav
│   │   └── Navigation.js
│   ├── redux
│   │   ├── store.js
│   │   └──reducers ..
│   └── screens
│       ├── HomeScreen.js
│       └── more screens..
└── package.json
```

  
## Start;

use npm install in project folder.

```bash
  npm install
```

start project

```bash
  npx expo start
```
